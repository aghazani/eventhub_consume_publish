# Install the App

`yarn` or `npm install`

# Create a new .env file at the root

### Add these entries to the file

```properties
CONSUMER_GROUP = ...
CONNECTION_STRING = ...
EVENT_HUB_NAME = ...
```

### and replace the ... by the correct value

# Start a DEV server

`yarn dev`

# Build

`yarn build`

This is just a demo app, nothing here is optimized :)
